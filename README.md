# Student Information Card

A simple Python program that displays a formatted student information card in the terminal.  
The program demonstrates basic programming concepts while practicing professional Git workflows such as branching, committing, and pull requests.

## Features

- Display student name (original)
- Display major and graduation year (original)
- Display graduation year (original)
- Display email address (added)
- Display hometown (added)
- Display favorite course (added)
- Improved formatting with borders and emojis
- Clear comments explaining program purpose

## Technologies Used

- Python 3.8+
- Git for version control
- GitHub for repository hosting

## Installation and Setup

### Prerequisites
- Python 3.8 or higher
- Git installed
- GitHub account

### Steps to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/najsahar/CPSC4205_GitWorkflow.git
2. Navigate to the project directory:
   cd CPSC4205_GitWorkflow
3. Run the program:
   python student_info.py

## Usage

Simply run the program to view the formatted student information card in the terminal.
